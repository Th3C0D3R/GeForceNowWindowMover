# Code of Conduct

## Our commitment
We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected behavior
- Be respectful and constructive.
- Assume good intent and discuss ideas, not people.
- Accept feedback gracefully.

## Unacceptable behavior
- Harassment, insults, or discriminatory language.
- Personal attacks, trolling, or intimidation.
- Publishing private information without permission.

## Enforcement
Project maintainers may remove, edit, or reject comments, commits, issues, or pull requests that violate this Code of Conduct.

## Reporting
If you experience or witness unacceptable behavior, open a private report by creating a GitHub issue and marking it as conduct-related in the title.
